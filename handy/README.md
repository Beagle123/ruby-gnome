# Ruby/Handy

Ruby/Handy is a Ruby binding of Handy.

## Requirements

* Ruby/GTK4 in [Ruby-GNOME](https://ruby-gnome.github.io/)
* [Handy](https://gnome.pages.gitlab.gnome.org/libhandy/)

## Install

    gem install handy
