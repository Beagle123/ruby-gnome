# Ruby/VTE4

Ruby/VTE4 is a Ruby binding of VTE for use with GTK4.

## Requirements

* Ruby/GTK4 in
  [Ruby-GNOME](https://ruby-gnome.github.io/)
* [VTE](https://live.gnome.org/Terminal/VTE)

## Install

    gem install vte4
