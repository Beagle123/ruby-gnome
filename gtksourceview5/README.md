# Ruby/GtkSourceView5

Ruby/GtkSourceView5 is a Ruby binding of gtksourceview-5.x.

## Requirements

* Ruby/GTK4 in
  [Ruby-GNOME](https://ruby-gnome.github.io/)
* [GtkSourceView](https://wiki.gnome.org/Projects/GtkSourceView) 5.0.0 or later

## Install

    gem install gtksourceview5
